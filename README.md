# multi-agent-travel-planner-main
我构建了一个多 Agent 协作的智能旅游行程规划系统。它基于 6 个专业 Agent 分工完成用户偏好解析、目的地推荐、航班比价、酒店匹配、活动安排和预算校验，并通过 Pipeline 编排、并行搜索和预算循环机制自动生成完整旅行方案。项目同时提供 Python、Java、Go 三种实现版本，支持 CLI、FastAPI、Streamlit 前端和 REST API 调用，并配套单元测试与架构文档，能够模拟从需求输入到行程生成、预算超限调整、结果输出的完整闭环流程。相比传统手动规划方式，该系统可以显著提升行程规划效率，并展示了我对多 Agent 架构、异步并发、后端 API 设计和跨语言工程实现的综合能力。
